# L45 Robotics and Imitation Learning: RT-1, Gato, and SayCan

The real-world robotics problem is not control in the abstract. It is getting a policy to survive the kitchen.

## Summary
- Robotics foundation models need large offline data sets, high-capacity architectures, and language as glue.
- RT-1 is the concrete stack: EfficientNet plus TokenLearner, discretized actions, and 3 Hz inference.
- BC-Z and Gato are the main baselines; RT-1 wins on robustness across task and kitchen shifts.
- SayCan and Inner Monologue add planning and feedback on top of the policy stack.
- DIAL turns relabeling into a scaling trick by using VLMs to generate richer instructions.

## Non-Obvious Insights
- Diversity beats repetition: more task types matter more than more demos on the same task.
- Discretizing continuous actions is what makes the model fast and token-friendly.
- Teleop data is more homogeneous than wild play data, which is part of why RT-1 can scale.
- Robot learning is shifting from online RL to offline learning with decoupled data generation.
- The planner is only as good as the low-level skill buffet it can choose from.
- Six images of history is already a meaningful context bottleneck; long-horizon control still wants better memory.
