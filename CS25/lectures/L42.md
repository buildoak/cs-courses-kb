# L42 Transformers United: Attention as Sparse Distributed Memory

Public-source note: the playlist extract stops at 41 accessible items. I used the nearest public CS25 neuroscience lecture as the fallback source for this slot.

## Summary
- The lecture argues that attention can be read as a memory retrieval rule, not just a neural-network trick. Sparse distributed memory gives the geometric picture. Source: `0:06-7:09`.
- Softmax is not arbitrary in this framing. It behaves like a sharpened exponential weighting over nearby patterns, which approximates circle intersection in high-dimensional space. Source: `8:00-20:24`.
- The modern Hopfield view makes the transformer connection precise enough to be useful: queries, keys, values, and normalization all map cleanly onto the memory story. Source: `12:23-20:24`.
- The biological side matters, too. The Tolman-Eichenbaum machine ties transformer-style retrieval to hippocampal memory, grid cells, place cells, and path integration. Source: `1:13:21-1:18:20`.
- The end-state is a cognitive architecture argument: memory should be structured, context-sensitive, and efficient, not a flat dump of everything ever seen. Source: `1:16:16-1:22:03`.

## Key Concepts
- **Sparse Distributed Memory.** High-dimensional read/write memory with sparse occupancy.
- **Hamming distance.** The discrete metric used in the first approximation.
- **Circle intersection.** The geometric quantity that approximates attention weights.
- **Softmax.** The sharpening operator that makes attention behave like selective recall.
- **Modern Hopfield network.** The continuous memory view of attention.
- **Queries, keys, values.** The retrieval interface between current state and stored memory.
- **Grid cells.** The positional encoding analogue for path integration.
- **Place cells.** The memory-specific representation of location.
- **Hippocampus / MEC / LEC.** The biological substrate the talk maps onto.
- **Cognitive map.** The larger structure that memory and navigation build together.

## Non-Obvious Insights
- Attention looks heuristic until you see it as retrieval over a high-dimensional memory space.
- The softmax exponent is not just a convenience. It is the thing that makes attention selective enough to be useful.
- Biological plausibility here is not decoration. It gives a concrete story for why memory, normalization, and context gating belong together.
- Path integration is the cognitive-architecture analogue of positional encoding.
- The architecture only needs the memories that matter in the current context. Global recall is not the goal.
