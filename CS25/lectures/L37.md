# L37 Transformers United: Reasoning Models as Agents: Deliberative Alignment, Multimodal Intelligence, and Tool Use

Reasoning stops being a vibe once the model can act.

## Summary
- The public Stanford abstract frames reasoning models as agents that plan, decompose problems, use tools, and handle multimodal inputs like images and diagrams. Source: Stanford EE event page.
- The alignment problem shifts with agency: the target is not just correct answers, but helpful, honest, and harmless behavior under action.
- Hongyu Ren positions the talk around visual walkthroughs and code examples, which keeps the framing operational instead of purely theoretical.
- This fits the CS25 arc where reasoning, tools, and multimodality are being treated as one capability stack.

## Key Concepts
- Deliberative alignment.
- Tool use.
- Multimodal inputs.
- Planning and problem decomposition.
- General problem-solvers.
- Helpful, honest, harmless behavior.
- Reasoning algorithms for large language models.
- O-mini reasoning systems.

## Non-Obvious Insights
- Reasoning here is an orchestration problem: plans, tools, and modality handling, not just chain-of-thought text.
- Once the model can act, alignment becomes an end-to-end systems problem instead of a postprocessing filter.
- Images and diagrams are not a side channel here. They are part of the same reasoning loop.
- The talk is about making models behave like general problem-solvers, which is a stricter target than benchmarked answer quality.
