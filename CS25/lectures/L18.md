# L18 Attention as Sparse Distributed Memory: Cerebellum, Cognitive Maps, and Transformers

This lecture is really two talks. The first argues that softmax attention is a form of sparse distributed memory; the second shows that hippocampal-entorhinal memory can be written as a transformer-like model with modern Hopfield networks. Source: `[8:14]` to `[25:19]`, `[43:27]` to `[1:18:47]`.

## Summary
- The first half starts from the attention equation and argues that query-key-value attention is an exponential weighting scheme over memory, which lines up with sparse distributed memory. Source: `[8:14]` to `[16:23]`.
- The speaker makes the biological claim concrete by mapping SDM onto the cerebellum, where granule cells, Purkinje cells, and climbing fibers implement read/write behavior. Source: `[26:25]` to `[33:43]`.
- The talk keeps the short-term memory point explicit: this is not only a long-term storage story. Attention is the pattern-based view of SDM, while the neuron-based view is the biological one. Source: `[34:10]` to `[37:45]`.
- The second speaker introduces the Tolman-Eichenbaum machine, which separates structure from content to model the hippocampal-entorhinal system. Source: `[43:27]` to `[1:00:59]`.
- The model uses grid cells for structure, LEC for content, hippocampus for binding, and a modern Hopfield network that behaves like attention over memories. Source: `[1:01:05]` to `[1:18:47]`.

## Key Concepts
- **Attention equation.** Queries match keys, softmax sharpens the weights, and values are aggregated into the next representation. Source: `[8:14]` to `[12:05]`.
- **Sparse Distributed Memory.** SDM stores and recalls patterns by activating a subset of neurons in a high-dimensional space. The read/write intersection decays exponentially with distance. Source: `[14:46]` to `[16:23]`.
- **Beta coefficient.** The softmax temperature maps to the SDM Hamming-distance selectivity. The lecture uses fitted beta values as weak evidence that transformer attention is doing the "right" thing. Source: `[16:10]` to `[23:32]`.
- **Cerebellar circuit.** Mossy fibers, granule cells, Purkinje cells, and climbing fibers implement separate activation and update paths. Source: `[26:25]` to `[33:43]`.
- **Grid cells and place cells.** The second talk leans on classical spatial navigation findings: place cells, grid cells, and entorhinal structure. Source: `[48:24]` to `[51:44]`.
- **Tolman-Eichenbaum machine.** The model factorizes structure and memories, using path integration for location and Hopfield-style memory for content binding. Source: `[59:00]` to `[1:18:47]`.
- **Modern Hopfield networks.** The memory layer behaves like attention over stored patterns, with softmax sharpening similarity and reconstructing the closest memory. Source: `[1:14:21]` to `[1:16:11]`.

## Non-Obvious Insights
- The softmax is not a cosmetic detail. The exponential weighting is the thing that makes the SDM analogy work. Source: `[14:17]` to `[16:23]`.
- The lecture's biological story is surprisingly specific: separate read and write lines, a huge fan-in at the Purkinje cell, and a direct route to a plausible memory circuit. Source: `[27:15]` to `[33:43]`.
- The short-term versus long-term memory split is cleaner than it first looks. The pattern-based transformer view is the short-term memory view; the neuron-based SDM view is the brain implementation. Source: `[34:47]` to `[37:45]`.
- The transformer and the cognitive map model share the same core move: split structure from content. In TEM, grid cells encode structure and the hippocampus binds it to observations. Source: `[59:00]` to `[1:02:06]`.
- The main systems advantage of the TEM-style factorization is scale. Adding context to the memory model no longer explodes the representation the way the naive outer-product version does. Source: `[1:18:00]` to `[1:18:20]`.
