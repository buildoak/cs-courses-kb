# L05 Automatic Differentiation Implementation

Lecture 5 turns the AD idea into Needle internals: values, ops, cached arrays, lazy materialization, and graph-valued gradients.

## Summary
- The lecture starts from the homework shape and then walks the Needle codebase, where `Tensor` inherits from `Value` and stores both data and graph metadata. Source: transcript `[0:17]` to `[18:43]`.
- `cached_data`, `inputs`, `op`, and `requires_grad` are the core fields. Together they describe both the value and how the value was computed. Source: transcript `[11:02]` to `[18:43]`.
- `realize_cached_data` is the engine behind eager execution, while lazy execution defers materialization until a consumer asks for the value. Source: transcript `[34:16]` to `[44:53]`.
- `detach` and `stop_gradient` cut the history chain. That is how you stop a long-running training loop from preserving the whole graph forever. Source: transcript `[45:43]` to `[51:10]`.
- `Op.gradient` returns tensors, not raw arrays, so the backward pass is itself a computational graph and can support higher-order derivatives. Source: transcript `[54:48]` to `[1:05:04]`.

## Key Concepts
- **`Value` / `Tensor`.** The base object that stores both array data and graph provenance. Source: transcript `[11:02]` to `[16:38]`.
- **`Op`.** Operator classes encapsulate both forward computation and backward rule logic. Source: transcript `[16:40]` to `[18:43]`.
- **`cached_data`.** The realized numeric value of a node. Source: transcript `[23:32]` to `[24:18]`.
- **`requires_grad`.** Flag that says whether a value should receive gradient propagation. Source: transcript `[16:08]` to `[16:38]`.
- **`realize_cached_data`.** Lazy materialization path that recursively computes dependencies. Source: transcript `[36:44]` to `[41:26]`.
- **Eager vs lazy.** Two execution styles over the same graph. Source: transcript `[41:45]` to `[44:53]`.
- **`detach` / `stop_gradient`.** Graph cutoff for values that should stop carrying history. Source: transcript `[48:41]` to `[50:59]`.
- **Gradient function.** The local reverse-mode rule for one operator. Source: transcript `[54:48]` to `[1:00:56]`.
- **`gradient_as_tuple`.** The shape-normalizing wrapper that makes single- and multi-input ops fit the same implementation path. Source: transcript `[1:01:09]` to `[1:03:07]`.

## Non-Obvious Insights
- Needle is not just a tensor wrapper. It is a graph object with array payloads attached.
- Eager and lazy are not competing APIs. They are different tradeoffs on when to realize the same graph.
- The memory leak demo is structural, not accidental: if the loss accumulator keeps graph history, the whole chain stays alive.
- `detach` is the right escape hatch when you need a value but no longer want autograd history.
- Returning tensors from `gradient` is the key design choice that makes gradient-of-gradient possible without a special second system.
